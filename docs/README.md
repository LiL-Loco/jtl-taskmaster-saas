# JTL TaskMaster - Dokumentation

## Übersicht
JTL TaskMaster ist eine SaaS-Lösung zur Automatisierung von JTL-Wawi-Prozessen. Die Anwendung ermöglicht es Benutzern, verschiedene Aufgaben zu automatisieren und zu planen.
